字符串

在程序开头使用use utf8来支持Unicode字符。

单引号：字符串直接量

除了单引号'和反斜杠\以外，所有字符表示自身。

双引号：字符串直接量

双引号内的反斜杠可以转义控制字符，也可以八进制和十六进制表示任意字符，以及控制格式。

\x{2668}        #表示Unicode码位为2668的字符。
\cC                #表示Ctrl+C    
\l                #将下一个字母转换为小写
\u                #将下一个字母转换为大写
\L,\U            #将下面所有的字母，直到\E，全部转化为小/大写

字符串操作符

  1. 使用.(句点)来连接字符串。如："hello"."world"即"helloworld"。
  2. 使用x(小写x)来重复字符串。重复次数会去尾取整。如："hi"x(1+1)即"hihi"。

自动数字/字符串转换

自动转换，妈妈再也不用担心数据类型。

Perl内置警告信息

Warnings

使用use warnings来开启警告功能。也可以在运行时使用perl -w,或者在释伴行使用#!/usr/bin/perl -w来开启。

发出警告不意味着停止执行，perl会根据内置规则继续。

Diagnostics

使用use diagnostics来启用更加详细的警告。同样地，也可以用perl -Mdiagnostics。

标量变量

标量变量只存储一个变量，名称以$(美元符号)开头，大小写敏感。如果启用了utf-8，可以用合法的Unicode作为变量名。

$name
$A_VERY_LONG_NAME
$名字                #只有开启了utf可用。

赋值

与其他语言中赋值无异。可以使用双目运算符，例如：$a += 5将把a自增5。特别的，.也可以双目运算，作为追加字符串。

字符串中的标量变量内插

将字符串内出现的标量变量转换为它的值，例如：

$a = "hello $name"中的$name会被替换为它的值。如果此变量不存在，将会使用空白替代。

使用{}来包含变量名，避免Perl检测变量名时混淆。

chr()与ord()

借助代码点来创建字符。chr()使用代码点来创建字符，例如$a = chr(0x03C9)，ord()返回字符的代码点。

Perl操作符优先级与C语言相同

结合性 操作符

通常，使用括号更加方便。

对于数字和字符串，有不同的比较操作符。

比较 数字 字符串
相等 == eq
不相等 != ne
小于 < lt

大于或等于 >= ge

布尔值

没有布尔类型，依照规则判断。

  1. 如果是数字，0为假，其他为真。

  2. 如果是字符串，空字符串为假，其他为真。

  3. 其他数据先转换为数字或字符串，再判断。

    undef为假，引用为真。

获取用户输入

<STDIN>操作符：获取用户的整行输入，包括换行符。使用chomp()来去掉换行符。

chomp($text = <STDIN>);        #也可以简写为<>
print $text;

undef值

如果没有赋值就使用某个标量的值，变量被定义为undef。**这并不会阻止程序的执行！**因此，应当尽量使用-w开启警告。

define函数

要判断某个字符串是undef还是空字符串，使用define函数。

其他

括号

perl是形式自由的，只要不影响语义，括号可以不写。但是，while,if等控制结构的括号必须写！

类型转换

使用诸如int()来进行类型转换。

列表和数组

列表和数组是perl中表示复数的方式。数组中的每一个元素都是单独的，有序的标量变量。perl中数组元素从0递增。

定义列表

  1. 列表直接量。使用()来定义列表，元素之间使用空格分割。

    例如:(1,2,3,4,"hello")，或者(1..100)。后者定义的是从1到100的整数列表。

    ..是范围操作符，只能向上不能向下。

  2. qw简写

    qw是一个操作符。后接括号定义，不过不用加引号。

    例如：qw( a b c d e f g );

    圆括号是界定符，不一定是圆括号，所以，下面的方法也可以：

    qw! a b c d !;
qw& a b c d &;
qw{ a b c d };
qw[ a b c d ];

    qw中的字符，可以使用\来转义。

访问数组元素

赋值

  1. 使用下标来访问数组元素。注意要加上$符号： 
$item[0]="hello";
$item[1]=100;    
$item[3.14-1]='Go!';    #数字表达式直接去除小数部分

​ 没有赋值，则自动变成undef.。数组会自动增加长度，不会抛出错误！

  1. 多个元素操作

    ($a,$b)=("a","b");
($list_a[0],$list_a[1])=($list_a[1],$list_a(0));    #互换变量

  2. @字符

    使用@来代表整个列表。因此，可以使用以下方式给列表赋值：

    @list=qw(a b c d);
@another=(@list,"data",500);
@array=5..9;

最后一个元素，以及倒数

  1. 使用$#来取得最后一个元素。

    $last=$#items;

  2. 使用负值索引。超出则为undef。

    $last=$items[-1];

列表操作

1. pop和push

  1. pop操作符：取出最后一个元素并返回。

    $item=pop(@items);    #括号可以不写

  2. push操作符：在列表末尾添加元素。

    push(@array,0);
push @array,1..10;
push @array,@another_array;

2. shift和unshift

和pop与push相反，这两者在列表的开头作用。

  1. shift操作符：在列表左边弹出元素。
  2. unshift操作符：在列表左边添加新元素。

3. splice

splice用来对数组进行删除。它有多种用法：

@array=qw[ a b c d e f g];
@removed=splice @array,1;        #删除第一个以及以后的元素，即删除b,c,d,e,f,g
@removed=splice @array,1,2;        #删除从第一个开始的两个元素，即删除b,c
@removed=splice @array,1,2,1..2;    #删除从第一个开始的两个元素，并替换为1,2
@removed=splice @array,1,0,qw(hello) ;#通过把第三参数设为0，实现在列表任意位置插入子列表

4.reverse

reverse用来反向操作一个列表。

@list=reverse @list;                #反向一个列表
@list=reverse 1..7;                #输出7..1
foreach (reverse 1..5){}

遍历数组和列表

foreach $循环变量名 (@数组名){    #也可以使用qw直接产生
    ...
}

注意：循环体内的变量修改对数组无效！在离开循环体后，改动会自动还原!

默认变量$_

在foreach结构中的循环变量可以省略，默认为$_

foreach (1..10){    #循环变量为$_
    ...
}

事实上是通用的默认变量
